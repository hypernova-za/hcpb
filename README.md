# hcpb

Simple console progress bar for Golang which shows the following info
- Progress bar
- Position
- Maximum value
- Rate
- Elapsed time
- ETA time

```
[>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>] 79.6k/79.6k @ 267/s in 04:58 ETA 00:00
```

It's buggy and incomplete and I don't recommend that you only use it for educational purposes
